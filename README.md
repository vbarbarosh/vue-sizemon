Monitoring size of a [positioned ancestor](https://www.w3.org/TR/css-position-3/#comp-abspos).

    <div class="rel">
        <vue-sizemon v-model="size" />
        <div>
            width={{ size.w }}<br>
            height={{ size.h }}
        </div> 
    </div>

## Installation

    npm i @vbarbarosh/vue-sizemon
